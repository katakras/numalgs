# Numerical Algorithms library with Python bindings

## Introduction

This is a small project with numerical algorithms implemented in C++ with bindings to be used in Python. Efficiency is one of the pillars of the design.

## Development instructions (with VSCode)
We use the following extensions:
- C/C++
- C++ TestMate
- Clang-Format
- CMake Tools
- Test Adapter Converter
- Test Explorer UI
